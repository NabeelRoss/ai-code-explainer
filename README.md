# AI Code Explainer

An intelligent web application that analyzes code snippets and explains their logic in plain English.

## Features
- **AI-Powered Analysis:** Uses the `Salesforce/codet5-base-multi-sum` model for high-quality code summarization.
- **Modern UI:** Built with Tailwind CSS, featuring a glassmorphism design and responsive layout.
- **Adjustable Depth:** Users can control the length of the explanation (Concise vs. Deep Dive).

## Tech Stack
- **Backend:** Flask, PyTorch, Transformers
- **Frontend:** HTML5, Tailwind CSS (CDN), Vanilla JS
- **Model:** CodeT5 (Base Multi-Sum)
